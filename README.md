## 🎮 About the Game

**Earth Invaders** is a simple 2D arcade-style game inspired by classic space shooters.

The player controls a character/ship and must defend against incoming enemies while trying to survive as long as possible.

---

## 🎯 Features

- 👾 Enemy spawning system  
- 🔫 Shooting mechanics  
- 💥 Collision detection  
- 🎵 Sound effects and/or music  
- 🎨 Custom graphics and assets  
- ⏱️ Score or survival-based gameplay  

---

## 🕹️ Gameplay

- Move your character to avoid enemies  
- Shoot to eliminate threats  
- Survive as long as possible  
- Try to achieve the highest score  

---

## 🖥️ How to Run

1. Navigate to the project folder:

```bash
cd earth_invaders
```

2. Run the game:

```bash
python code.py
```

---

## 🎮 Controls

Example (may vary depending on implementation):

- `← / →` – movement  
- `SPACE` – shoot  
- `ESC` – exit game
